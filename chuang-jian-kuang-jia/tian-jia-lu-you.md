# 添加路由 - 新建PHP框架框架

**参考资料**

* https://github.com/noahbuscher/Macaw
* https://github.com/klein/klein.php

**创建路由文件夹及文件**

```
 mkdir -p framework/routing
 touch route.php
```

**路由类基本组成**

* 一个存放路由的属性
* 一个设置路由属性的方法
* 还需要一个调度方法 , 调度url的请求与响应
