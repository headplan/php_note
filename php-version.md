# 版本特性

#### PHP5.3

**新特性**

1.支持命名空间\(Namespace\)

2.支持延迟静态绑定\(Late Static Binding\) - 后期静态绑定

3.支持goto语句

4.支持闭包,Lambda/Anonymous函数

5.新增两个魔术方法\_\_callStatic\(\)和\_\_invoke\(\)

6.新增Nowdoc语法

7.在类外也可使用const来定义常量

8.三元运算符增加了一个快捷书写方式

9.HTTP状态码在200-399范围内均被认为访问成功

10.支持动态调用静态方法
